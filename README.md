# Generador de Key

Aplicación React + Vite (en español) inspirada en un generador de keys con tareas de verificación.

## Uso

```bash
npm install
npm run dev
```

Para compilar producción:

```bash
npm run build
npm run preview
```

## Personalización

- Edita los enlaces de verificación en `src/App.jsx` (array `TAREAS`).
- Ajusta duración del key y del temporizador en las constantes `DURACION_HORAS` y `EXPIRA_SEGUNDOS`.
- Estilos en `src/styles.css`.
